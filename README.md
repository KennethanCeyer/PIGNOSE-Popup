PIGNOSE-Popup
=============

*PIGNOSE Layer popup plugin*
Lightweight and simple modal popup.

###Usage
1. import ```jquery library``` ([Download link](http://www.jquery.com/download/)).
2. import ```PIGNOSE-Popup.js```.
3. just type ```$select.pignosePopup();```.
